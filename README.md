# Build with:
# Django
# Bootstrap


Steps on Linux:
- python3 -m venv .venv
- source .venv/bin/activate
- python3 -m pip install -r requirements.txt
- python manage.py runserver


Steps on Windows (Gitbash):
- python -m venv .venv
- source .venv/Scripts/activate
- python -m pip install -r requirements.txt
- python manage.py runserver
